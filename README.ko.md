## 특징
Whisper Mate는 OpenAI의 Whisper-IA 모델을 사용하여 오디오 파일 및 비디오 파일을 대량으로 텍스트로 변환할 수 있습니다. 내장된 자막 편집기를 사용하여 변환 결과를 섹션별로 표시할 수 있습니다.
모든 변환 프로세스는 기기 내에서 로컬에서 실행되며 개인정보가 보호됩니다.
