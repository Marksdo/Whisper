## 🔑 主な特長
- **プライバシーを最優先**: 文字起こし・翻訳・エクスポートを完全ローカルで実行。機密データが Mac の外へ出ることはありません。
- **最適なエンジンを選択**: FluidAudio、whisper.cpp、WhisperKit、Apple のオンデバイス AI（macOS 26+）、Kokoro English Speech Synthesis を用途に合わせて切り替え。
- **ライブ制作ワークフロー**: 録音、字幕、翻訳、フローティングプレビュー、オートメーションチェーンをリアルタイムで監視。
- **仕上がりそのままのプレビュー**: 内蔵プレーヤーで字幕の書体・サイズ・行間をレスポンシブに確認。
- **拡張可能なツール群**: テンプレート、JavaScript プロセッサー、辞書、正規表現変換を組み合わせて高度な字幕整形を自動化。
- **字幕専用デスク**: 編集、レビュー、バッチ入出力、Final Cut Pro / QuickLook / GIF / MKV パイプラインまでワンストップで管理。

## 🔥 主要機能
- **バッチ & ライブ文字起こし**: 大量メディア、ライブマイク入力、アプリ音声ルーティング、同時翻訳まで対応。
- **字幕・翻訳エディタ**: 結合、分割、タイミング調整、Quick-Cut、置換、キーボード中心の操作を提供。
- **多言語翻訳**: DeepL、Apple Translate、GPT、Gemini、DeepSeek、OpenRouter、Ollama 互換のローカル LLM を利用可能。
- **自動化プレイブック**: ジョブ完了後にスクリプト、エクスポート、要約、通知を自動実行。
- **入出力エコシステム**: SRT、VTT、ASS、FCPXML、GIF、PDF、DOCX、MKV、オーディオクリップなどをシームレスに扱う。
- **ビジュアルツール**: フローティングウィンドウ、柔軟なフォント・行間設定、テキストのみ・分割ビューを即座に切替。
- **LLM 連携**: Apple オンデバイスモデル、Ollama、リモート LLM と接続し、字幕の磨き上げ・要約・スクリプト修正に活用。
- **プロジェクト管理 & スナップショット**: プロジェクトのグループ化、キュー優先度設定、リバーシブルなスナップショット保存、安全なアーカイブ、リモートフォルダ監視。
- **動画ポストプロダクション**: Quick Cut 波形編集、チャプター、シーンキーフレーム、GIF/MP4 エクスポート、ハードサブ埋め込み。

## 🚀 ユースケース
- 映画、ポッドキャスト、インタビューの字幕制作・翻訳・ナレーション。
- 会議、授業、ライブ配信向けリアルタイムメモと要約。
- 多言語バッチを自動化して処理するメディアチーム。
- スナップショットで音声コーパスをアーカイブ・注釈付けする学術/研究チーム。
- 長尺動画をショート、GIF、ナレーション付きクリップへ再構成するマーケ/ソーシャルチーム。
- 高品質な監査ログが必要なサポート、営業、法務向け録音のコンプライアンス対応。
