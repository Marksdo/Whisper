## 🔑 核心優勢
- **隱私優先的設計**：轉錄、翻譯與輸出全數在本機執行，敏感素材不會離開你的 Mac。
- **彈性切換最佳引擎**：於 FluidAudio、whisper.cpp、WhisperKit、Apple 裝置端 AI（macOS 26+）與 Kokoro English Speech Synthesis 之間自由切換，應付各類配音需求。
- **即時製作流程**：即時監控錄音、字幕、翻譯、浮動預覽與自動化鏈結。
- **所見即所得預覽**：在內建播放器檢視字幕字體、字級與行距，並自適應調整排版。
- **可擴充工具鏈**：結合範本、JavaScript 處理器、字典與正規表示式轉換，自動化複雜的字幕整理。
- **專屬字幕工作桌**：集中管理編輯、審核、批次匯入匯出，以及 Final Cut Pro / QuickLook / GIF / MKV 全流程。

## 🔥 核心能力
- **批次＋即時轉錄**：處理大量素材、即時麥克風擷取、應用程式音訊導向與同步翻譯。
- **字幕與翻譯編輯器**：支援合併、拆分、重新計時、Quick-Cut、批次取代與完整鍵盤操作。
- **多語翻譯**：串接 DeepL、Apple Translate、GPT、Gemini、DeepSeek、OpenRouter 或本地 Ollama 相容 LLM。
- **自動化劇本**：任務完成即自動觸發腳本、輸出、摘要或通知。
- **多元匯入／匯出生態**：無縫處理 SRT、VTT、ASS、FCPXML、GIF、PDF、DOCX、MKV、音訊片段等格式。
- **視覺化工具**：浮動視窗、可調字型與行距，一鍵切換純文字或分割檢視。
- **LLM 協作**：連結 Apple 裝置端模型、Ollama 或遠端 LLM，進行潤稿、摘要或腳本修正。
- **專案控管與快照**：專案分組、佇列優先權、可回復快照、安全封存與遠端資料夾監控。
- **影音後製**：Quick Cut 波形剪輯、章節、場景關鍵影格、GIF/MP4 匯出與硬字幕編碼。

## 🚀 典型情境
- 電影、Podcast、訪談的字幕製作、翻譯與配音。
- 會議、課程、直播的即時筆記與摘要。
- 媒體團隊以自動化快速處理多語批次任務。
- 學術或研究團隊運用快照封存並標註語料庫。
- 行銷與社群團隊將長影片重製成 Shorts、GIF 或旁白剪輯。
- 客服、銷售或法務錄音符合稽核需求並保留高品質追蹤。
