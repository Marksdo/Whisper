## 🔑 核心优势
- **隐私优先的设计**：转录、翻译与导出全部在本地运行，敏感素材不会离开你的 Mac。
- **随时切换最佳引擎**：在 FluidAudio、whisper.cpp、WhisperKit、Apple 本地 AI（macOS 26+）以及 Kokoro English Speech Synthesis 之间自由切换，胜任配音工作。
- **实时制作流程**：实时监控录音、字幕、翻译、浮动预览与自动化链路。
- **所见即所得预览**：在内置播放器中检视字幕字体、字号与行距，响应式调节排版。
- **可扩展工具链**：组合模板、JavaScript 处理器、词典和正则转换，自动化复杂字幕清理。
- **专用字幕工作台**：集中管理编辑、审核、批量导入导出及 Final Cut Pro / QuickLook / GIF / MKV 全流程。

## 🔥 核心能力
- **批量 + 实时转录**：处理海量素材、实时麦克风采集、应用音频路由与同步翻译。
- **字幕与翻译编辑器**：支持合并、拆分、重新计时、Quick-Cut、批量替换与全键盘操作。
- **多语种翻译**：接入 DeepL、Apple Translate、GPT、Gemini、DeepSeek、OpenRouter 或本地 Ollama 兼容 LLM。
- **自动化剧本**：任务完成后自动触发脚本、导出、摘要或通知。
- **全能导入/导出生态**：无缝处理 SRT、VTT、ASS、FCPXML、GIF、PDF、DOCX、MKV、音频剪辑等格式。
- **可视化工具**：浮动窗口、可调字体与行距，一键切换纯文本或分屏模式。
- **LLM 协作**：连接 Apple 本地模型、Ollama 或远程 LLM，完成润色、摘要或脚本修复。
- **项目控制与快照**：项目分组、队列优先级、可回滚快照、安全归档与远程文件夹监控。
- **视频后期**：Quick Cut 波形编辑、章节、场景关键帧、GIF/MP4 导出与硬字幕编码。

## 🚀 典型场景
- 电影、播客、访谈的字幕制作、翻译与配音。
- 会议、课程、直播的实时笔记与摘要。
- 媒体团队通过自动化快速处理多语言批任务。
- 学术或研究团队使用快照归档并注释语料库。
- 市场与社媒团队将长视频重制为短视频、GIF 或配音剪辑。
- 满足合规要求的客服、销售或法律录音，提供高质量审计追踪。
